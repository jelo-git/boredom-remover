# boredom-remover

Project for shitty games that will keep you awake during university lectures.


## Installation

Install boredom-remover with npm

```bash
  npm install
```
    
## Running

Use following command

```bash
  nodemon index.js
```
or 
```bash
  node index.js
```

Application will be available at: http://localhost/
